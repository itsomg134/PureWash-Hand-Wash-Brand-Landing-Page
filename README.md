# PureWash – Hand Wash Brand Landing Page

A clean, modern, and responsive landing page for a premium hand wash brand. Built with HTML & CSS — no frameworks, no bloat.

---

## Features

-  **Hero section** with product highlight and call-to-action
-  **Feature badges** (plant-based, dermatologist tested, eco-packaging)
-  **Product grid** displaying 4 hand wash variants
-  **Testimonial** block for social proof
-  **Newsletter subscription** CTA
-  **Fully responsive** – works on mobile, tablet, and desktop
-  **Clean, warm aesthetic** with a nature-inspired color palette
-  **Google Fonts** (Inter) and **Font Awesome** icons

---

## Tech Stack

- **HTML5**
- **CSS3** (Flexbox, Grid, custom properties)
- **Font Awesome** (icons)
- **Google Fonts** (Inter)

> No JavaScript, no external libraries – just pure HTML & CSS.

## Customization

- **Colors** – Edit the CSS variables in the `<style>` section (e.g., `#1e2b2c`, `#3b8b7a`)
- **Products** – Modify the `.product-card` blocks to change names, prices, or icons
- **Hero image** – Replace the `<i class="product-icon">` with an actual `<img>` tag if you have a bottle graphic
- **Testimonial** – Update the text and author name inside the `.testimonial` section

---

## Responsive Breakpoints

- **Desktop-first** – scales down gracefully
- **Tablet & mobile** – stack layout, smaller typography, adjusted padding
